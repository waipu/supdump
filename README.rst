supdump - Supplementary code dump
================================
|Build Status|

Package for supplementary code used in some projects.
Renamed from sup, because a project<https://github.com/frnsys/sup>_ with similiar contents already exists.

Dependencies
------------
net/multinet:
- pycurl
- adispatch
special:
- simplejson
